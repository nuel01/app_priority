USER REVIEWS PRIORITIZATION SYSTEM

Online customer feedbacks have evolved into a useful and increasingly important base for information about customer requirements, product characteristics, requirement reuse for the production of new systems, the implementation of new features, bug fixing, or the improvement of existing features or functionalities, improving user experience in a timely manner, providing a new and critical channel to extract user feedback, and a significant base for measuring the success of an Information System. User reviews can reveal a variety of information, including user thoughts or attitudes about specific items, user complaints, new feature requests, and observations about existing software features. Because these assessments are sometimes lengthy and unclear, developers frequently require methods for identifying and extracting requirements from them before selecting in which order they should be addressed.

What it Does The goal of this system is to prioritise user reviews from mobile applications on google play store using developer-centric features. To make use of the website, first input the App ID of the selected app, select the preferred method to search, select the aspect for ranking then select the number of reviews to be prioritised and click enter. The reviews will be pre-processed and categorised into ten categories. The elicited ranking information on the requirements from developers' will be used in producing a ranked prioritised requirement data, which produces a prioritised user reviews in form of a document ready for download .
